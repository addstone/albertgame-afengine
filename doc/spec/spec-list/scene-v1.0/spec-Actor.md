# 对象名称
## Actor 实体

# 对象介绍
实体，是表示一个游戏对象，用以自动化管理自身的单位，一个实体由一系列数据和一系列控制行为的实体组件组成，并且包含子实体。此外，一个游戏对象也包含一个变换Transform用以表示空间位置。
