# 对象名称
## BehaviorBeanComponent 行为组件 继承ActorComponent

# 对象介绍
行为组件表示一个实体的一系列行为方式。

# 对象功能

## 添加实体行为ActorBehavior
## 获取指定名称的实体行为ActorBehavior
## 删除指定名称的实体行为
